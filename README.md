## 📚 Overview

**AI Expense Tracker** is a web application designed to help users track daily expenses, manage budget efficiently, and leverage AI for categorization and insights. With a simple, attractive UI and insightful analytics, this tool empowers anyone to take control of their finances.

---

## ✨ Features

- **Add, Edit, Delete Transactions**  
  Easily log new income or spending entries, and manage your transaction history.

- **Expense Visualization**  
  Dynamic charts and summaries show spending by category, recent activity, or over time.

- **Smart Categories (AI-Powered)**  
  Optional: Uses AI to auto-tag and recommend categories for new expenses.

- **Monthly Budgets & Goals**  
  Set custom budgets for different categories and track progress.

- **Responsive Design**  
  Fully mobile-friendly, modern UI with thoughtful interactions.

- **Filtering & Search**  
  Quickly filter transactions by date, category, or keyword.

## 🖥️ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/farhankhan0986/AI-Expense-Tracker-.git
   cd AI-Expense-Tracker-
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Start the development server**
   ```bash
   npm start
   # or
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
